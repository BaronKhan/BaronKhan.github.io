J.E.F.F - Raspberry Pi Robot

November 2016 | Group | University

Overview

A Raspberry Pi robot written in Python and capable of waypoint navigation, wall-following, place recognition and object-finding. This robot uses several advanced robotics algorithms, such as Monte-Carlo Localisation (MCL) and Occupancy Mapping.

Given three obstacles and a known map of the environment, the robot is capable of locating these objects and bumping into them before returning to its starting position. This is done using MCL (Monte-Carlo Localisation) for accuracy in navigation, and sensors for detecting an object and walls (which are already known as a map of the environment is provided beforehand).

Features

- Wall Following
- Place Recognition
- Waypoint Navigation (using MCL)

Hardware

- Raspberry Pi
- BrickPi Hat
- Lego NXT sensors and actuators

TIMELINE - Washing Line Timer

March 2017 | Group | University

Overview

Attach to a washing line to start a timer for your clothes to dry which changes dynamically based on maximum wind speed and temperature. The device consists of a temperature sensor and an airflow sensor rotating on a gear motor. It sends JSON-formatted updates to a PC server every 30 seconds, which then updates the estimated the time for the clothes to dry. Also includes a rain sensor to alert the user when it is raining.

Hardware

- Adafruit Feather HUZZAH ESP8266
- D6F-V MEMS Air Flow Sensor
- Adafruit ADS1115 4-Channel ADC Breakout
- SI7021 Temp + Humidity Sensor
- Servo Motor
- Rain Sensor

Software

- Micropython
- Eclipse Paho MQTT

MIPS GNU C Compiler

March 2016 | Individual | University

Overview

A partial compiler which outputs MIPS assembly code that can be assembled and linked using the GCC MIPS toolchain Written in C++ and supports expressions, control flow and function calls. Also features a sizeable amount of error handling. Utilises tools such as flex for tokenization and bison for parsing.

Features

- Integer type variables
- All assignments and expressions
- Basic control flow (if, while, for loop)
- Function calls (max. 4 arguments)
- Global variables
- Integer arrays
