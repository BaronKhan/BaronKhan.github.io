Hey, Spark!

The Hey, Spark! project aims to add voice recognition functionality to the Cisco Spark Boards from scratch, to allow multiple users to execute different commands on the Spark Board using just their voice, such as starting or ending a call, returning to the dashboard, creating new chatrooms, and more.

This project uses the Nvidia TensorRT API - a deep learning inference optimizer and runtime engine - to perform the "Hey Spark" detection on the GPU, and the Houndify cloud service for the voice requests.
